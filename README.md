## Name
SVT batch verifier tests for C code in XDP / eBPF.


## Description
Contains test batches with vulnerable C XDP / eBPF programs to test the verifier capabilites, the XVTLAS is the tool for the tests automation and result collection.
The programs folder contains the eBPF programs and the associated metadata files, that are used by the tool to auto launch and test.
