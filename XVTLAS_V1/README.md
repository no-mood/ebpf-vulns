## To build the tool 

### Requirements :
- Golang

### Steps : 
`go build -o xvtlas .`

### Running : 
Example : 

`sudo  ./xvtlas --export "./output/" --kernel "6.8.58" --path "./template_folders/" --pretty "./pretty-verifier/pretty_verifier.py" --verbose --interactive`

For more use: 
`xvtlas --help`
